Observations and Scenes (scarlet.observation)¶

The observation module defines the Scene, the meta data associated with a scarlet source, and the Observation class, which contains all of the necessary information to transform a model into an observed frame.

Reference/API¶

class scarlet.observation.LowResObservation(images, wcs=None, psfs=None, weights=None, filtercurve=None, padding=3, target_psf=None, structure=None)[source]¶

Data and metadata for a set of observations to resample on a different grid

Attributes:
images: array or tensor

3D data cube (bands, Ny, Nx) of the image in each band. These images must be resampled to the same pixel scale and same target WCS (for now). wcs: WCS object World Coordinate System associated with the images.

psfs: array or tensor

PSF for each band in images.

weights: array or tensor

Weight for each pixel in images. If a set of masks exists for the observations then then any masked pixels should have their weight set to zero.

padding: int

Number of pixels to pad each side with, in addition to half the width of the PSF, for FFT’s. This is needed to prevent artifacts due to the FFT.

structure: array

An array that encodes the position of the images of this observation into a high resolution (spatial and spectral) scene.
