Sources (scarlet.source)¶

It is recommended to initialize sources by inheriting from the Component class and overwriting the __init__ and update methods to initialize and constrain sources respectively.

In general, neither the SED nor the morphology are known ahead of time so scarlet offers a set of convenience classes, to generate an initial morphology and SED for a source at a given location in the image: * PointSource * ExtendedSource * MultiComponentSource

class scarlet.source.MultiComponentSource(sky_coord, scene, observations, bg_rms, flux_percentiles=None, obs_idx=0, thresh=1, symmetric=True, monotonic=True, **component_kwargs)[source]¶

Create an extended source with multiple components layered vertically. Uses ~scarlet.source.ExtendedSource to define the overall morphology, then erodes the outer footprint until it reaches the specified size percentile. For the narrower footprint, it evaluates the mean value at the perimeter and sets the inside to the perimeter value, creating a flat distribution inside. The subsequent component(s) is/are set to the difference between the flattened and the overall morphology. The SED for all components is calculated as the best fit of the multi-component morphology to the multi-band image in the region of the source.

Parameters:
flux_percentiles: list

The flux percentile of each component. If flux_percentiles is None then flux_percentiles=[25], a single component with 25% of the flux as the primary source.

See `ExtendedSource` for a description of the parameters

class scarlet.source.PointSource(sky_coord, scene, observations, symmetric=False, monotonic=True, center_step=5, delay_thresh=10, **component_kwargs)[source]¶

Extended source intialized with a single pixel

Point sources are initialized with the SED of the center pixel, and the morphology of a single pixel (the center) turned on. While the source can have any constraints, the default constraints are symmetry and monotonicity.
