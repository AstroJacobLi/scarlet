Cache (scarlet.cache)¶

scarlet.cache contains the Cache class, which contains frequently needed values that can be time consuming to create and relatively inexpensive to store, such as the indices needed for monotonicity, PSF FFTs, etc.

Reference/API¶

class scarlet.cache.Cache[source]¶

Cache to hold all complex proximal operators, transformation etc.

Convention to use is that the lookup name refers to the class or method that pushes content onto the cache, the key can be chosen at will.
