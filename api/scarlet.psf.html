scarlet.psf¶

class scarlet.psf.PSF(X, shape=None)[source]¶

Bases: object

Class to represent PSFs

Parameters
X: array-like or method

If X is an array, it represent an image of the PSF in every band. If X is a callable method, it describes a function that can generate a PSF when given coordinates (y, x).

shape: tuple

Shape of the 2D image to generate an PSF image for. Only used if X is a method.

image
